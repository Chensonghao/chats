# Chats
> nodejs + express + mongodb

> socket.io

> 前端支持两种架构：angularjs 或 react-redux + es6

> webpack打包,支持热加载

### Start
##### 安装依赖
```
    npm install
```
##### 运行mongodb
```
    mongod --dbpath [your DB path]
```
##### 启动程序
前端默认使用react-redux
```
    npm start
```
如果前端使用angularjs
```
    npm run angular
```

本应用使用的是gravatar头像，如果期望使用email登陆进去显示自定义头像的话，请确保你的email账户在[gravatar](http://cn.gravatar.com/)上传过头像。
