# Chats
> 使用socket.io＋angularjs 搭建的聊天应用、使用webpack打包、支持热加载

### Start
##### 安装依赖
```
    npm install
```
##### 运行mongodb
```
    mongod --dbpath [your DB path]
```
##### 启动程序
前端默认使用reactjs
```
    npm start
```
如果前端使用angularjs
```
    npm run angular
```

本应用使用的是gravatar头像，如果期望使用email登陆进去显示自定义头像的话，请确保你的email账户在[gravatar](http://cn.gravatar.com/)上传过头像。
